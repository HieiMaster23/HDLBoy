# Test ROMs

This directory holds project-owned Game Boy ROM images used for FPGA bring-up.

## `minimal_led_blink.gb`

Generated by:

```text
python scripts/generate_minimal_led_rom.py
```

Purpose:

- 32 KiB no-MBC `ROM ONLY` cartridge image;
- jumps from both `0x0000` and the standard cartridge entry point `0x0100` to
  the same program at `0x0150`;
- repeatedly writes `0x05` and `0x0A` to `0xFF80`, allowing
  `sdram_cpu_rom_top` to prove that the CPU is executing code fetched from
  SDRAM by blinking the LED pattern.

Recommended loader smoke command:

```text
quartus_stp -t scripts/load_rom_virtual_jtag.tcl --dry-run --max-bytes 32 roms/minimal_led_blink.gb
```
